# Django RPG Project

## Getting Started

To run the development server, follow these steps:

1.  **Activate the virtual environment:**

    ```bash
    . venv/bin/activate
    ```

2.  **Run the development server:**

    ```bash
    python3 manage.py runserver
    ```

The application will be available at `http://127.0.0.1:8000/`.
